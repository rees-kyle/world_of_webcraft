Typography

The art and technique of arranging type to make written language readable and visually appealing.

code

            
                .output {
                    font-family: 'Courier New', Courier, monospace;
                    font-size: 18px;
                    font-weight: 600;
                    font-style: italic;
                    margin: 20px;
                }   

                .aligned-right {
                    text-align: right;
                }

                .line-through {
                    text-decoration: line-through
                }

                .capitalize {
                    text-transform: capitalize;
                }


                <div class="output">
                    <p>
                        This paragraph demonstrates the default font family (Courier New),
                        font size (18px), font weight (600), and font style (italic) applied to the body.
                    </p>
                    <p class="aligned-right">This text is aligned to the right.</p>
                    <p class="line-through">This text has a line through it.</p>
                    <p class="capitalize">this text is capitalized.</p>
                </div>

Output

This paragraph demonstrates the default font family (Courier New), font size (18px), font weight (600), and font style (italic) applied to the body.

This text is aligned to the right.

This text has a line through it.

this text is capitalized.
