Transitions and Animations

Transitions

Code

            
                .box-transition {
                    width: 100px;
                    height: 100px;
                    background-color: blue;
                    transition: width 2s, background-color 2s, transform 2s;
                }

                .box-transition:hover {
                    width: 200px;
                    background-color: red;
                    transform: rotate(45deg);
                }


                <div class="box-transition"></div>

Animations

Code

            
                @keyframes slideAndFade {
                    0% {
                        transform: translateX(0);
                        opacity: 1;
                    }
                    50% {
                        transform: translateX(100px);
                        opacity: 0.5;
                    }
                    100% {
                        transform: translateX(200px);
                        opacity: 1;
                    }
                }

                .box-animation {
                    width: 100px;
                    height: 100px;
                    background-color: green;
                    animation-name: slideAndFade;
                    animation-duration: 4s; /* Duration of the animation */
                    animation-timing-function: ease; /* Speed curve of the animation */
                    animation-delay: 1s; /* Delay before the animation starts */
                    animation-iteration-count: infinite; /* Animation repeats infinitely */
                    animation-direction: alternate; /* Animation direction alternates */
                    animation-fill-mode: both; /* Element retains styles from both the start and end of the animation */
                    animation-play-state: running; /* Animation is running */
                }


                <div class="box-animation"></div>
