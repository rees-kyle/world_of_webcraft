Float and Clear Properties

Float

The float property in CSS is primarily used to position elements horizontally within their containing element. It's commonly used for creating layouts where elements can be floated to the left or right, allowing other elements to flow around them, which is particularly useful for creating multi-column layouts or aligning images and text side by side.

Left

Code

            
.float-left {
    float: left;
    margin-right: 0.625rem;
}

Floating an element in CSS removes it from the normal document flow, which can lead to unexpected layout behaviors, especially on smaller devices where space is limited. These behaviors may cause content to overlap or disrupt the intended layout. To address this issue, we use a technique called clearing floats, which ensures that subsequent content flows as intended. You can learn more about clearing floats further down this page.

None

Setting float: none; in CSS removes any floating behavior applied to an element.

Code

            
.float-none {
    float: none;
}


<img src="../../images/float.jpg" alt="A rubber duck floating on water" width="250" height="250" class="float-none">

Clear

Used to control the behavior of an element concerning floated elements that come before it in the HTML markup. When you apply the clear property to an element, you're essentially telling it to move below any floated elements that appear before it.

Left

Code

            
.clear-left {
    clear: left;
    background-color: rgb(74, 190, 105);
}


<div class="container">
    <div class="example-container">
        <img src="../../images/float.jpg" alt="A rubber duck floating on water" width="250" height="250" class="float-left">

Others

  • clear: right; The element will move below any floated elements that are floated to the right.
  • clear: both; The element will move below any floated elements regardless of their direction (left or right).
  • clear: none; The element will not move below any floated elements. This is the default value.
