Colors and Backgrounds

Properties

Color

Name

Code
            
                .name {
                    color: red;
                }

                <p class="name">Set the color using the name like red.</p>
Output

Set the color using the name like red.

Hex Code

Code
            
                .hex-code {
                    color: #00ff99;
                }

                <p class="hex-code">Represents colors using a six-character (#RRGGBB) hexadecimal code.</p>
Output

Represents colors using a six-character (#RRGGBB) hexadecimal code.

RGB

Code
            
                .rgb {
                    color: rgb(0, 0, 250);
                }

                <p class="rgb">Define colors by specifying the intensity of Red, Green, and Blue, each ranging from 0 to 255.</p>
Output

Define colors by specifying the intensity of Red, Green, and Blue, each ranging from 0 to 255.

RGBA

Code
            
                .rgba {
                    color: rgba(0, 0, 250, 0.5);
                }

                <p class="rgba">Define colors by specifying the intensity of Red, Green, Blue, and an Alpha channel for opacity, each ranging from 0 to 255 for RGB and 0 to 1 for Alpha.</p>
Output

Define colors by specifying the intensity of Red, Green, Blue, and an Alpha channel for opacity, each ranging from 0 to 255 for RGB and 0 to 1 for Alpha.

HSL

Code
            
                .hsl {
                    color: hsl(0, 100%, 50%);
                }

                <p class="hsl">Define colors by specifying Hue (0 to 360), Saturation (0% to 100%), and Lightness (0% to 100%).</p>
Output

Define colors by specifying Hue (0 to 360), Saturation (0% to 100%), and Lightness (0% to 100%).

HSLA

Code
            
                .hsla {
                    color: hsla(0, 100%, 50%, 0.5);
                }

                <p class="hsla">Define colors by specifying Hue (0 to 360), Saturation (0% to 100%), Lightness (0% to 100%), and Alpha for opacity (0 to 1).</p>
Output

Define colors by specifying Hue (0 to 360), Saturation (0% to 100%), Lightness (0% to 100%), and Alpha for opacity (0 to 1).

Background

Color

Code
            
                .background-color {
                    background-color: green;
                }

                <div class="background-color">Green background color.</div>
Output
Green background color.

Image

Code
            
                .image {
                    background-image: url('../../images/mountains.jpg');
                    background-size: cover;
                    background-position: center;
                    padding: 100px 100px 85px;
                    color: white;
                    text-align: center;
                }

                <div class="image">Text with a background image.</div>
Output
Text with a background image.
