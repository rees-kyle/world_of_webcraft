Responsive Web Design

Responsive Web Design (RWD) makes websites look good and work well on all devices, like desktops, tablets, and phones. It adjusts the layout and content automatically to fit different screen sizes.

Code

                
/* Adjustments for mobile */
@media screen and (max-width: 45.5rem) {
    body {
        font-size: 1.125rem;
        line-height: 1.4;
    }

    h1, h2, h3, h4, h5, h6 {
        padding-left: 1rem;
    }

    h1 {
        font-size: 1.75rem;
    }

    h2 {
        font-size: 1.5rem;
    }

    h3 {
        font-size: 1.375rem;
    }

    h4 {
        font-size: 1.25rem;
    }

    h5 {
        font-size: 1.125rem;
    }

    h6 {
        font-size: 1rem;
    }

    p {
        padding: 1.125rem;
        padding-left: 1rem;
        padding-bottom: 1rem;
    }

    small {
        padding: 1rem;
    }

    #footer {
        padding: .2rem;
        padding-left: .5rem;
    }

    #website-title {
        padding: 1rem;
        font-size: 1.75rem;
    }

    nav {
        padding: 0.5rem 1rem;
        padding-bottom: 4.5rem;
    }

    .menu > li, .menu > a {
        display:block;
        padding-left: unset;
        text-align: center;
    }

    .menu {
        width: 12.5rem;
    }

    .submenu {
        position: unset;
        width: 10rem;
        padding-bottom: 2.5rem;
    }

    .submenu2 {
        list-style-type: none;
        padding: 0;
        margin: 0;
        width: 8rem;
        display: none;
        position: absolute; /* position relative to parent submenu item */
        left: 100%; /* position to the right of the parent 'submenu' */
        top: 0;
    }

    .menu a {
        padding-top: .46875rem;
        padding-bottom: .46875rem;
    }

    code {
        font-size: 1.125rem;
        margin-left: unset;
    }

    .video-container {
        padding-bottom: 56.25%; /* 16:9 aspect ratio */
        max-width: 100%;
    }
}
