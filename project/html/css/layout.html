Display

Determines how an element is displayed on the page.

Block

Code
            
.block {
    display: block;
    width: 200px;
    height: 100px;
    background-color: green;
    margin: 20px auto;
    text-align: center;
    line-height: 100px;
    border: 0.09375rem solid #000000;
}

<div class="container">
    <div class="example-container">
        <div class="block">Block Element 1</div>
        <div class="block">Block Element 2</div>
        <div class="block">Block Element 3</div>
    </div>
</div>
Output
Block Element 1
Block Element 2
Block Element 3

Inline

Code
            
.inline {
    display: inline-block;
    background-color: green;
    padding: 5px;
    margin-right: 10px;
    border: 0.09375rem solid #000000;
}

<div class="container">
    <div class="example-container">
        <span class="inline">Inline Element 1</span>
        <span class="inline">Inline Element 2</span>
        <span class="inline">Inline Element 3</span>
    </div>
</div>
Output
Inline Element 1 Inline Element 2 Inline Element 3

Position

Specifies the positioning method of an element.

Relative

Code
            
.parent {
    width: 200px;
    height: 200px;
    background-color: rgb(255, 255, 124);
}

.relative {
    position: relative;
    top: 50px;
    left: 30px;
    width: 100px;
    height: 100px;
    color: black;
    font-weight: bold;
    background-color: rgb(255, 121, 121);
    border: 0.09375rem solid #000000;
}


<div class="container">
    <div class="example-container">
        <div class="parent">
            <div class="relative">Relative</div>
        </div>
    </div>
</div>
Output
Relative

Element will be positioned relative to its original position.

Absolute

Code
            
.parent {
    width: 200px;
    height: 200px;
    background-color: rgb(255, 255, 124);
}

.absolute {
    position: absolute;
    width: 100px;
    height: 100px;
    color: black;
    font-weight: bold;
    background-color: rgb(119, 255, 119);
    border: 0.09375rem solid #000000;
    box-shadow: 0.625rem 0.625rem 0.0625rem rgba(0, 0, 0, 0.5);
}


<div class="container">
    <div class="example-container">
        <div class="parent">
            <div class="absolute">Absolute</div>
        </div>
    </div>
</div>
Output
Absolute

Element will be positioned relative to the closest positioned ancestor and is removed from the document flow.
