Properties

Display

Determines how an element is displayed on the page.

Code

            
.block {
    display: block;
    width: 200px;
    height: 100px;
    background-color: green;
    margin: 20px auto;
    text-align: center;
    line-height: 100px;
}

<div class="block">Block Element 1</div>
<div class="block">Block Element 2</div>
<div class="block">Block Element 3</div>

Output

Block Element 1
Block Element 2
Block Element 3

Code

            
.inline {
    display: inline;
    background-color: green;
    padding: 5px;
    margin-right: 10px;
}

<span class="inline">Inline Element 1</span>
<span class="inline">Inline Element 2</span>
<span class="inline">Inline Element 3</span>

Output

Inline Element 1 Inline Element 2 Inline Element 3

Position

Specifies the positioning method of an element.

Relative

Code
            
.container {
    position: relative;
    width: 300px;
    height: 300px;
    background-color: lightblue;
    border: 1px solid #000;
    margin-bottom: 20px;
}

.parent {
    width: 200px;
    height: 200px;
    background-color: yellow;
}

.relative {
    position: relative;
    top: 50px;
    left: 30px;
    width: 100px;
    height: 100px;
    background-color: lightcoral;
}


<div class="container">
    <div class="parent">
        <div class="relative">Relative</div>
    </div>
</div>
Output
Relative
  • The child element moves relative to where it would normally be within the parent.
  • It still affects the layout of other elements within the parent.

Absolute

Code
            
                .absolute {
                    position: absolute;
                    top: 50px;
                    right: 20px;
                    width: 100px;
                    height: 100px;
                    background-color: lightgreen;
                }


                <div class="container">
                    <div class="parent">
                        <div class="relative">Relative</div>
                    </div>
                </div>
Output
Absolute
  • The child element is positioned relative to the .parent element.
  • It does not affect the layout of other elements within the parent, as it is taken out of the normal document flow.
