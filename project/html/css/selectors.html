Selectors

Universal

Code

            
* {
    margin: 0;
    padding: 0rem;
    box-sizing: border-box;
}

Type (element)

Code

            
p {
    font-size: 1rem;
    padding: .625rem;
    padding-left: 2rem;
    padding-bottom: 2rem;
}

Class

Code

            
.text-label {
    font-size: 1.25rem;
    font-weight: bold;
    padding-right: 1rem;
}

ID

Code

            
    #website-title {
        padding: 1rem;
        font-size: 1.75rem;
    }

Attribute

Code

            
input[type="radio"] {
    height: 1.5rem;
    width: 7rem;
    accent-color: green;
}

Descendant

Code

            
.radio-container p {
    position: relative;
    font-weight: bold;
    top: 2rem;
    margin-bottom: 1rem;
    left: -1rem;
}

Child

Code

            
.menu > li, .menu > a {
    display: inline-block;
    padding-left: .625rem;
}

Adjacent Sibling

Code

            
h4 + p {
    text-decoration: underline;
}

<div class="container">
    <h4>Fourth-Level Heading</h4>
    <p>
        This paragraph is a sibling of a fourth-level heading.
        The adjacent sibling combinator selects an element that is directly adjacent to another element.
        In other words, it targets the immediate next sibling of a specified element.
    </p>
    <p>The next element does not get styled.</p>
</div>

Output

Fourth-Level Heading

This paragraph is a sibling of a fourth-level heading. The adjacent sibling combinator selects an element that is directly adjacent to another element. In other words, it targets the immediate next sibling of a specified element.

The next element does not get styled.

General Sibling

Code

            
h5 ~ p {
    text-decoration: underline;
}

<div class="container">
    <h5>Fifth-Level Heading</h5>
    <p>
        This paragraph is a general sibling of a fifth-level heading. 
        The general sibling combinator selects all elements that are siblings of a specified element and come after it,
        not necessarily immediately.
    </p>
    <p>The next element also gets styled.</p>
</div>

Output

Fifth-Level Heading

This paragraph is a general sibling of a fifth-level heading. The general sibling combinator selects all elements that are siblings of a specified element and come after it, not necessarily immediately.

The next element also gets styled.
