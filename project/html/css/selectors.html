Selectors

Universal

Code

            
                * {
                    color: gray;
                }

Type (element)

Code

            
                p {
                    color: red;
                }

Output

Class

Code

            
                .my-class {
                    text-decoration: underline;
                }

                <p class="my-class">This is another paragraph.</p>

Output

Code

                    font-style: italic;
                }

                <p id="my-id">This is a third paragraph.</p>

Output
