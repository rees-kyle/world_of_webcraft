Selectors

Universal

Code

            
                * {
                    color: gray;
                }

Type (element)

Code

            
                p {
                    color: red;
                }

Output

This is a paragraph.

Class

Code

            
                .my-class {
                    text-decoration: underline;
                }

                <p class="my-class">This is another paragraph.</p>

Output

This is another paragraph.

ID

Code

            
                #my-id {
                    font-style: italic;
                }

                <p id="my-id">This is a third paragraph.</p>

Output

This is a third paragraph.

Attribute

Code

            
                input[type="text"] {
                    border: 1px solid black;        
                }

                <input type="text" placeholder="Type something...">

Output

Descendant

Code

            
                div p {
                    background-color: green;
                }

                <div>
                    <p>This paragraph is inside a div.</p>
                </div>

Output

This paragraph is inside a div.

Child

Code

            
                ul > li {
                    list-style-type: square;
                }

                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                </ul>

Output

  • Item 1
  • Item 2

Adjacent Sibling

Code

            
                h4 + p {
                    margin-top: 50px;
                }

                <h4>Fourth-Level Heading</h4>
                <p>This paragraph is a sibling of a fourth-level heading.</p>

Output

Fourth-Level Heading

This paragraph is a sibling of a fourth-level heading.

General Sibling

Code

            
                h5 ~ p {
                    font-size: 32px;
                }

                <h5>Fifth-Level Heading</h5>
                <p>This paragraph is a general sibling of a fifth-level heading</p>

Output

Fifth-Level Heading

This paragraph is a general sibling of a fifth-level heading
