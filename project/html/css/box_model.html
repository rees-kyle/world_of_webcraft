Box Model

Demonstration

Code

            
.box {
    width: 100%;
    max-width: 61rem;
    padding: 1.25rem;
    border: .3125rem solid darkgreen;
    margin: 0 auto;
    background-color: black;
    color: white;
    font-size: 1.125rem;
    font-family: monospace;
}

<div class="container">
    <div class="box">
        - The text is the 'content'. <br>
        <br>
        - The 'width' property specifies the width of the content area; of the element. <br>
        <br>
        - The 'padding' property specifies the width on both the left and right sides of the content. <br>
        <br>
        - The 'border' is in dark green. <br>
        <br>
        - The 'margin' is the outside spacing. <br>
        <br>
        - The 'background-color' is in black. <br>
    </div>
</div>

Output

  • The text is the 'content'.
  • The 'width' property specifies the width of the content area; of the element.
  • The 'padding' property specifies the width on both the left and right sides of the content.
  • The 'border' is in dark green.
  • The 'margin' is the outside spacing.
  • The 'background-color' is in black.
