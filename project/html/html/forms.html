Forms

Used to collect user input.

Elements

Text Input

Code

                
<div class="container">
    <div class="input-container">
        <label for="username" class="text-label">Username:</label>
        <input
            type="text"
            id="username"
            name="username"
            required
            minlength="3"
            maxlength="12"
            pattern="[A-Za-z0-9]+"
            title="Username should be alphanumeric and at least 3 characters long."
        >
    </div>
</div>

Output
