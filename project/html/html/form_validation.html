Form Validation

Used to ensure accurate and properly formatted user input, enhance user experience, and prevent invalid data from reaching the server.

Attributes

Pattern

Code

                
                    <p>Postcode:</p>
                    <input 
                        type="text" 
                        name="postcode" 
                        pattern="[A-Za-z]{1,2}[0-9Rr][0-9A-Za-z]?\s[0-9][ABD-HJLNP-UW-Zabd-hjlnp-uw-z]{2}"
                        title="Enter a valid UK postcode."
                        placeholder="e.g. AB1 2CD"
                    >

Output

Postcode:
