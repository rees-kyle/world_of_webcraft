Form Validation

Used to ensure accurate and properly formatted user input, enhance user experience, and prevent invalid data from reaching the server.

Attributes

Pattern

Code

                
<div class="container>
    <div class="pattern-container">
        <label for="postcode">Postcode:</label>
        <input 
            type="text"
            id="postcode"
            name="postcode" 
            pattern="[A-Za-z]{1,2}[0-9Rr][0-9A-Za-z]?\s[0-9][ABD-HJLNP-UW-Zabd-hjlnp-uw-z]{2}"
            title="Enter a valid UK postcode."
            placeholder="e.g. AB1 2CD"
        >
    </div>
</div>

Output

Min and Max

Code

                
<div class="container">
    <div class="input-container">
        <label for="age" class="age-label">Age:</label>
        <input type="number" id="age" name="age" min="18" max="123">
    </div>
</div>

Output

Length

Code

                
<div class="container">
    <div class="input-container">
        <label for="username" class="text-label">Username:</label>
        <input type="text" id="username" name="username" required minlength="3" maxlength="12" pattern="[A-Za-z0-9]+" title="Username should be alphanumeric and at least 3 characters long.">
    </div>
</div>
