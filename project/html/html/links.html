Links

Local Pages

Links to other pages within website.

Code

            
                <a href="index.html">Project Overview</a>

Output

Project Overview

Sections

Internal links to sections within the same page.

Code

            
                <p>Jump to <a href="#email">email</a> section.</p>
                
                
                <section id="email">

Output

Jump to email section.

New Tab

Code

            
                <p>This <a href="https://www.example.com" target="_blank">link</a> opens in a new tab.</p>

Output

This link opens in a new tab.

Images

Embeds a hyperlink within the image.

Code

            
                <a href="https://www.example.com">
                    <img src="../../images/click_here.jpg" alt="Example Image" width="200" height="200">
                </a>

Output

Email

Code

            
                <p>Contact us via <a href="mailto:info@example.com">email</a>.</p>

Output

Contact us via email.
