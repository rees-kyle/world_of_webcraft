External

Embed multimedia content from external sources.

Youtube

Code

            
                <iframe 
                    width="560"
                    height="315"
                    src="https://www.youtube.com/embed/F6PqxbvOCUI?si=2JjW6obT-q_22kJx"
                    title="YouTube video player"
                    frameborder="0"
                    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
                    referrerpolicy="strict-origin-when-cross-origin"
                    allowfullscreen>
                </iframe>
