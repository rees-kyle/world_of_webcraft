Markup

Code Snippets

HTML entities are used within code snippets to ensure that these characters are displayed as text rather than interpreted as part of HTML markup.

Bold and Italics

Code

              
                  <p>
                      <b>This sentence is in bold and is used purely for stylistic purposes.</b>
                  </p>
                  <p>
                      <strong>
                          This sentence is also in bold but 
                          it indicates that the text has stressed emphasis or subtle importance,
                          affecting how assistive technologies interpret and read the text.
                      </strong>
                  </p>
                  <p>
                      <i>This sentence is in italics and is used purely for stylistic purposes.</i>
                  </p>  
                  <p>  
                      <em>
                          This sentence is also in italics but 
                          it indicates that the text has stressed emphasis or subtle importance,
                          affecting how assistive technologies interpret and read the text.
                      </em>
                  </p>

Output

This sentence is in bold and is used purely for stylistic purposes.

This sentence is also in bold but it indicates that the text has stressed emphasis or subtle importance, affecting how assistive technologies interpret and read the text.

This sentence is in italics and is used purely for stylistic purposes.

This sentence is also in italics but it indicates that the text has stressed emphasis or subtle importance, affecting how assistive technologies interpret and read the text.

Strikethrough

Code

                
                    <p>This is a <s>strikethrough</s> line.</p>

Output

This is a strikethrough line.
